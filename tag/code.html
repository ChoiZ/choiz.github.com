Elément cliqué en js

Pour savoir quel élément est cliqué j'utilise ce petit bout de javascript :

function checkclick(e) {
    e = e || window.event;
    var o = e.target || e.srcElement;
    console.log(o);
}
document.onclick=checkclick;

Protection de fichier tel qu'un htaccess en PHP

Posté le ven. 11 février 2011 • Tags : php, auth basic, code • Laissez un commentaire

Faire un système de protection (tel que htaccess) direct dans un fichier PHP :

<?php
if (!empty($_SERVER["PHP_AUTH_USER"]) &&
$_SERVER["PHP_AUTH_USER"] == "login" &&
$_SERVER["PHP_AUTH_PW"] == "password") {
    // code protégé ici
} else {
    header('WWW-Authenticate: Basic realm="Restricted area"');
    header('HTTP/1.0 401 Unauthorized');
    exit('Access Denied');
}

Attention l'utilisation du htaccess n'est pas recommandée …
