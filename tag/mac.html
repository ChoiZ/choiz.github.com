Installer jshint sur mac

Posté le lun. 14 octobre 2013 • Tags : jshint, mac, setup, npm, nodejs

Installation via npm de jshint (-g pour installer pour tous les utilisateurs) :

sudo npm install -g jshint

Faire un lien symbolique pour pouvoir utiliser directement jshint sans préciser tout le chemin a chaque fois :

ln -s /usr/local/share/npm/bin/jshint /usr/local/bin/jshint

Suppression des .DS_Store

Posté le mar. 05 juillet 2011 • Tags : Mac .DsStore, Mac, .DS_Store

Dans une console sur votre mac tappez :

defaults write com.apple.desktopservices DSDontWriteNetworkStores true

Redemarrez le mac.
