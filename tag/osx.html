Voir les fichiers cachés sur Lion

Posté le mer. 02 novembre 2011 • Tags : file, osx • Laissez un commentaire

Dans un terminal :

defaults write com.apple.Finder AppleShowAllFiles TRUE

killall Finder

Ouvrir une fenêtre du Finder et vous voyez les fichiers cachés.

Développement web avec Safari

Posté le lun. 05 septembre 2011 • Tags : dev, code, osx • Laissez un commentaire

Pour activer l'inspecteur web tapez dans le terminal :

defaults write com.apple.Safari WebKitDeveloperExtras -bool true

Redémarrez Safari et maintenant vous pouvez inspecter les éléments de vos pages web.
