Il suffit d’utiliser git show.

La commande :

git show branch_name:filename

Exemple :

git show reset_password:/templates/reset_password.tmpl

Installation de Firefox sur Debian

Posté le ven. 21 mars 2014 • Tags : Firefox, Debian • Laissez un commentaire

Télécharger la dernière version de Firefox sur le site de mozilla.

Décompresser l'archive dans :

/opt/firefox/

Créer un lien symbolique avec :

ls -sf /opt/firefox/firefox /usr/bin/firefox
