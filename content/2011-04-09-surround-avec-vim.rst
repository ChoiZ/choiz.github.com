J'ai ajouté quelques plugins dans ma configuration de Vim dont :
surround.vim.

.. raw:: html

   </p>

Ce plugin permet d'ajouter des balises ou tout autre élément autour d'un
texte.

.. raw:: html

   </p>

Comment l'utiliser ?

.. raw:: html

   </p>

``VS<b>`` permet d'ajouter les balises <b> et </b> avant et apres la
ligne courante et d'indenter celle-ci.

.. raw:: html

   </p>

``vims<a>`` permet d'ajouter le tag <a></a> autour du mot sur lequel
vous vous trouvez.

.. raw:: html

   </p>

``cst<b>`` permet de changer le tag <a> (change surround tag) par <b>

.. raw:: html

   </p>

``dst`` permet de supprimer le tag

.. raw:: html

   </p>
